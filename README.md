# GunZ dumps

these are dumped with https://github.com/glmcdona/Process-Dump by triggering the "fail to load directx" messagebox to dump the process while it is suspended. this allows me to dump fxp for example as it runs before its driver gets ran, which strips handles.  

i will try to maintain this repo as much as possible to make sure all the dumps are up to date  

if the dumps are outdated, make an issue or pm me on discord: "maiet"
